# 📋 Release Notes - OntoWave

## Overview

OntoWave is a lightweight JavaScript library for creating interactive documentation from Markdown with multilingual support, Prism syntax highlighting, Mermaid diagrams, and PlantUML rendering.

**Current Version**: v1.0.0  
**Last Updated**: December 2025

## 📦 Installation

### Via NPM

```bash
npm install ontowave
```

### Via CDN

```html
<!DOCTYPE html>
<html>
<head>
  <title>OntoWave Documentation</title>
</head>
<body>
  <script src="https://unpkg.com/ontowave@latest/dist/ontowave.min.js"></script>
  <script>
    // Configuration and initialization
    window.ontoWaveConfig = {
      sources: { en: 'README.md' },
      defaultLocale: 'en'
    };
  </script>
</body>
</html>
```
