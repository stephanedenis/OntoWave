Display Math

The integral of a Gaussian function:

$$ \int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi} $$

The sum of the first n natural numbers:

$$ \sum_{i=1}^{n} i = \frac{n(n+1)}{2} $$

$$ e^{i\pi} + 1 = 0 $$

🔍 Search Plugin

Full-text search across all documentation.

How to use:

  • Press / key to focus search
  • Type at least 2 characters to see results
  • Click on a result to navigate
  • Press Escape to close

The search plugin indexes all page content and provides instant results with highlighted matches.

Try searching for: "plugin", "mermaid", "math", "diagram"

📚 TOC Plugin (Table of Contents)

Auto-generates a table of contents from page headings.

Features:

  • Fixed position on the right side
  • Smooth scroll navigation
  • Active section highlighting (scroll spy)
  • Collapsible with toggle button
  • Auto-hides if less than 3 headings

Look at the right side of your screen to see the TOC in action! 👉

📊 Analytics Plugin

Google Analytics integration for page tracking.

This plugin tracks page views and navigation events. Configure with your GA tracking ID.

✨ Syntax Highlighter Plugin

Enhanced code block styling with custom features.

Example: JavaScript


// OntoWave plugin initialization
function initializePlugins(plugins) {
  return plugins.map(plugin => {
    console.log(`Loading plugin: ${plugin.name}`)
    return plugin.initialize()
  })
}

const plugins = [
  mermaidPlugin,
  mathPlugin,
  searchPlugin
]

Promise.all(initializePlugins(plugins))
  .then(() => console.log('All plugins ready! 🎉'))
  .catch(err => console.error('Plugin error:', err))

⚙️ Plugin Configuration

Configure plugins via window.ontoWaveConfig:


window.ontoWaveConfig = {
  plugins: [
    {
      name: 'mermaid',
      enabled: true,
      config: {
        theme: 'default',
        logLevel: 'error'
      }
    },
    {
      name: 'math',
      enabled: true
    },
    {
      name: 'plantuml',
      enabled: true,
      config: {
        serverUrl: 'https://www.plantuml.com/plantuml',
        format: 'svg'
      }
    },
    {
      name: 'search',
      enabled: true,
      config: {
        minChars: 2,
        maxResults: 10,
        placeholder: '🔍 Rechercher...'
      }
    },
    {
      name: 'toc',
      enabled: true,
      config: {
        position: 'right',
        sticky: true,
        minHeadings: 3
      }
    },
    {
      name: 'analytics',
      enabled: false,
      config: {
        trackingId: 'UA-XXXXX-Y'
      }
    },
    {
      name: 'custom-syntax-highlighter',
      enabled: true
    }
  ]
}
