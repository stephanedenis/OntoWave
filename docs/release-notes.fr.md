# 📋 Notes de Version - OntoWave

## Vue d'ensemble

OntoWave est une bibliothèque JavaScript légère pour créer de la documentation interactive à partir de Markdown avec support multilingue, coloration syntaxique Prism, diagrammes Mermaid et rendu PlantUML.

**Version Actuelle** : v1.0.0  
**Dernière Mise à Jour** : Décembre 2025

## 📦 Installation

### Via NPM

```bash
npm install ontowave
```

### Via CDN

```html
<!DOCTYPE html>
<html>
<head>
  <title>OntoWave Documentation</title>
</head>
<body>
  <script src="https://unpkg.com/ontowave@latest/dist/ontowave.min.js"></script>
  <script>
    // Configuration et initialisation
    window.ontoWaveConfig = {
      sources: { fr: 'README.md' },
      defaultLocale: 'fr'
    };
  </script>
</body>
</html>
```
